simple steps to recover grub in ubuntu
  • Boot Live CD
  • Open Terminal and Type sudo grub
  • new type the following command : "find /boot/grub/stage1"
  • If u get an error "File not found" then try : "find /boot/grub/stage1"
  • Output look like this : (hd0,1)-->> "harddisk + boot partition numbers(my /boot is at /dev/sda7, which translates to hd0,6 for grub)"
  • Set it as a root device :" grub> root (hd0,1)" press enter
  • Install Grub : "grub> setup (hd0)"
  • Exit Grub : "grub> quit"
  • Now restart ur system
  • Hurrey !!! you just installed your grub :)
