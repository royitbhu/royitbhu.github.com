1. simple steps on how to change the password using single user mode
  1. Reboot the Ubuntu
  2. When grub is loading,press 'Esc'
  3. Choose 'Ubuntu kernel(recovery mode)'
  4. Press 'e' to edit the kernel parameter
  5. Append 'single init=/bin/bash' to the kernel parameter
  6. Press 'b' to boot from that particular kernel

    7. You will enter single user mode . Your hard drive will be in read-only mode. Remount it in read-write mode using following command.

  7. "$ mount -o remount,rw /dev/sda1 "(drive name in which ubuntu is installed)
  8. Change your passwd : "$ passwd user"
  9. Reboot your machine and enter newly changed password
  10. Hurrey !!! Password has been changed
