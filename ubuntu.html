Some Important stuffs for Linux Geeks

1 .Install libgraph library in GCC (Graphics.h)

Step #1.Install the following in Synaptic Package Manager

  1. libsdl-image1.2
  2. libsdl-image1.2-dev
  3. guile-1.8
  4. guile-1.8-dev

Step #2. Download and install libgraph then

  1. ./configure
  2. sudo make
  3. sudo make install

2. Install LAMP (Linux-Apache-Mysql-PHP) in a single command
  1. sudo apt-get install apache2 php5 libapache2-mod-php5 mysql-server php5-mysql
3 .Some of the things to do immediately after installing ubuntu on your machine:
  1. Linux DC++ Client : sudo apt-get install linuxdcpp
  2. Adobe Acrobat Reader : sudo aptitude install acroread acroread-plugins acroread-escrip
  3. Macromedia Flash : sudo apt-get install flashplugin-nonfree
  4. Acroread plugin for firefox : sudo aptitude install mozilla-acroread
  5. Installing Sun Java JRE : sudo aptitude install sun-java6-jre sun-java6-plugin sun-java6-fonts
4 .Install Gnome classic theme in 11.04+ (who dont like Gnome Unity)
  1. Command : sudo apt-get install gnome-session-fallback
  2. To make Gnome Classic as default : sudo /usr/lib/lightdm/lightdm-set-defaults -s gnome-classic
5 .Install all the packages/softwares from old ubuntu to newly installed Ubuntu)
  1. If you want to switch to newly released ubuntu and need all the package/software which are currently installed ubuntu.then you can do two things :
  2. 1. Normal upgrade: sudo apt-get dist-upgrade
  3. 2. Install new released ubuntu : for this you will have to do 3 things
  4. A. Get list of package selections from old ubuntu : dpkg --get-selections > package.txt
  5. B. Set package selections using file package.txt in newly installed ubuntudpkg --set-selections < package.txt
  6. C. Now install these package using following : sudo apt-get dselect-upgrade
